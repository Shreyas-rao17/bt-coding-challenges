# Coding Challenge 2: Program to calculate simple interest

Write a program that accepts principal amount, rate of interest, and time, then calculates and displays the simple interest.

## Requirements
- Accept principal (P), rate (R), time (T)
- Compute Simple Interest using formula: SI = (P * R * T) / 100
- Display the result
- Handle invalid inputs

## Solution Approach
- Use a function for calculation
- Support floating point inputs
- Clear output formatting

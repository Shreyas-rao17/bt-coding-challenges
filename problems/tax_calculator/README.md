# Tax Calculator Challenge

Build a tax calculation program for New Tax Regime (2023).

Features:
- Accept employee details and salary components
- Calculate gross and taxable income
- Compute tax with slabs, rebate, cess
- Generate detailed reports
- Input validation

Tax slabs: 0-3L:0%, 3-6L:5%, 6-9L:10%, 9-12L:15%, 12-15L:20%, >15L:30%
Rebate: 100% if taxable <=7L
Cess: 4% on tax
Standard deduction: 50,000

# Challenge 24: Fibonacci Series

Write a function that generates the Fibonacci series up to a given number N.

## Requirements

- The function should return a list of Fibonacci numbers up to and including the largest Fibonacci number that is less than or equal to N.
- If N is less than 1, return an empty list.
- Include input validation.

## Example

- Input: 21
- Output: [1, 1, 2, 3, 5, 8, 13, 21]
